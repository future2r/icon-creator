# Icon Creator

A desktop GUI application for assembling Windows ICO files from a set of PNG source images.

## What It Does

Icon Creator scans a directory for PNG images named with a size-based pattern (e.g. `application_16.png`, `application_32.png`, …), lets you choose which sizes to include, and assembles them into a single `.ico` file.

## Requirements

* **Java 25** or later

## Build

Use the **Maven Wrapper** to build the project.
